    This project involves developing a deep learning model to classify user comments into three categories: doubt, irrelevant, and feedback. The data is preprocessed by tokenizing and padding text, while the labels are one-hot encoded for compatibility with the neural network. After training, the model achieves reasonable accuracy and an evaluation is conducted using metrics like F1 score to measure its performance on unseen data.
